# Assets

Put personal files here and point to them from `content.js`.

Suggested structure:

```text
assets/
  files/
    resume.pdf
  images/
    profile.jpg
    project-one.jpg
```

Then update fields like:

```js
avatarImage: "assets/images/profile.jpg"
resumeUrl: "assets/files/resume.pdf"
```

The template works without images, but real project screenshots and a real profile photo make it feel much more complete.
